Install Java 8, Maven, appium-client.
Open Project in IntelliJ Idea as Maven project.


You can run tests in three ways:
1. For run one package:
In the project go to src/test/java, choose speedtestApp package and make right click on the folder and choose “Run”

2. For run all tests:
Click on the test file "testng.xml" and choose  “Run”.

3. Command on Terminal "mvn clean test"
